搭建Hexo博客

发表于 2015-05-03   |  

Hexo是一个任人打扮的小姑娘。

名赫

为什么搭建Hexo博客

  • 作为一名程序员，应该有自己持续更新的技术博客。一方面，能用来记录自己所学和所研究的东西；另一方面，博客也是一个很好的展示自己能力的工具。

  • 国内有很多网站如博客园、简书、CSDN等优秀的网站可以让我们写自己的技术博客。但是对于搞技术的人来说，很重视逼格这个东西。搭建自己的技术博客，当然逼格相对高一点。

  • 试用过Octopress和WordPress搭建自己的博客，总是感觉不够好用。后来了解到了Hexo，试了一下感觉简洁易用，速度也快，所以最终决定用Hexo搭建自己的博客。

  • Hexo有很多漂亮酷炫的第三方主题，对于外貌协会的人来说是福音。如果你有兴趣，还可以打造自己的主题。

  • Hexo支持Markdown语言书写。

什么是Hexo

简而言之就是通过Node.js实现的一种博客框架，想要详细了解可以自己去Google词条,或者进入Hexo的官网了解。

Hexo在全球最大的男性交友网站GitHub上拥有7000＋的Star，足见Hexo的人气之高。想看Hexo源码的同学，请移步hexojs/hexo.

搭建Hexo博客的简易教程

网上有很多详尽的搭建Hexo的教程，在这里大致记录一下我搭建的过程，想要快速建站的同学可以按照我的步骤走。对了，我是在Mac上搭建的，其他系统大家自己去找教程吧。

1.安装Homebrew

首先我们知道hexo是基于node.js的，所以必须安装node.js。安装node.js的方法很多，这里使用Homebrew安装，所以要有先安装它.

终端里面一句话搞定：

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

2.安装Git和Github设置

终端里输入：

brew install git

Git的操作和配置相信大家都已经非常了解，在这里不再赘述。

3.安装node.js

用Homebrew,终端里面一句话搞定：

brew insatll node

4.安装Hexo

首先，在终端里面输入：

npm install -g hexo

然后，执行init命令初始化hexo到你指定的目录，也可以直接cd到目标目录执行hexo init.

hexo init /Users/minghe/hexoBlog

至此，全部安装工作完成。

5.在GitHub上部署Hexo

  • 首先要在GitHub建立与你用户名对应的仓库，仓库名必须为your_user_name.github.io.比如我的就是codemozart.github.io.这个也是以后你博客的网址。

  • 修改配置文件_config.yml.在你的hexo按转目录下找到这个文件，打开，把deploy下的type后面改成git,repo后面改成你自己仓库的地址。如下图：

6.Hexo的一些基本操作

在Hexo官网的文档中，详细的介绍了Hexo的各种使用，在这里我说一些常用的。

  • 发布新文章

    cd 到你的init目录下，执行

    hexo new [layout] <title>

    如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。

    比如 hexo new 搭建Hexo博客

  • 生成静态页面

    cd 到你的init目录下，执行

    hexo generate

  • 启动本地服务器

    现在终端里面执行下面三句话：

    npm install hexo-renderer-ejs --save 

npm install hexo-renderer-stylus --save


npm install hexo-renderer-marked --save

然后执行：

hexo server

在浏览器输入网址http://0.0.0.0:4000,就可以看到自己的blog了。虽然很简陋，但好歹有了一个属于自己的小窝了。现在本地已经简单的设置好了，但是现在域名和服务器都是基于自己的电脑，接下来需要跟github进行关联。

  • 部署到GitHub上
hexo deploy

然后在浏览器上输入your_user_name.github.io(比如我的就是codemozart.github.io,你要记得把your_user_name改成自己的用户名)就可以进入你的个人博客了。

6.更换主题

Hexo有很多优秀的主题，去官方提供的主题列表中选一个，按照里面的方法pull下来即可。

我的博客采用的是NexT主题，快速安装教程和进阶教程里面都有，读者自行点进去学习吧。

Markdown编辑器推荐

Hexo生成的文章是需要markdown语言书写的，市面上markdown的编辑器很多，我推荐两款自己用过的，觉得比较好的。

  • mac电脑，本地编辑器推荐Mou，简洁大方，实时预览。
  • 在线编辑器推荐Stackedit,十分强大，谁用谁知道
